# Lazar Minkov — Portfolio

A minimal, editorial personal website for an independent web developer in Bern.

## Preview locally

From this folder, run:

```sh
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173`.

## Structure

- `index.html` — home, about, portfolio and contact
- `blog.html` — expandable journal entries
- `styles.css` — full visual system and responsive layout
- `app.js` — navigation, motion, clock, cursor and contact behavior
- `assets/` — local fonts, favicon and original portfolio artwork
